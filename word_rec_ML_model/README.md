# 単語音声認識Webアプリケーション

## プロジェクト概要
このプロジェクトでは,以下の機能を持つアプリの製作を行った。複数の決められた単語の音声データを入力として与え、その音声データからメルスペクトグラムに変換。それを元に入力データが何の単語なのかを推定しその結果を出力。

## 機能
Recordのボタンを押し自分の音声を録音する。言い終わったタイミングでstopのボタンを押すことで音声をファイルとして保存できる。その音声ファイルから下の単語のうちどの音声に近いかを推定し、その候補の１番目から３番目までを確率の降順に出力される。

"_background_noise_", "backward", "bed", "bird", "cat", "dog", "down", "eight", "five", "follow", "forward", "four", "go", "happy", "house", "learn", "left", "marvin", "nine", "no", "off", "on", "one", "right", "seven", "sheila", "six", "stop", "three", "tree", "two", "up", "visual", "wow", "yes", "zero"




## モデル設計
pytorchを用いたCNNで画像認識を行う。

【ネットワーク構成】

入力→畳み込み層→Batch Normレイヤ→ReLU関数→プーリング層

→畳み込み層→Batch Normレイヤ→ReLU関数→プーリング層→平坦化

→全結合層→Batch Normレイヤ→ReLU関数→dropout→全結合層

【学習】

ミニバッチ学習で学習させる。その過学習を防ぐ工夫として以下を実装した。

1, 　入力データに対しDataAugmentationを行う。具体的にはホワイトノイズ、周波数マスキング(縦方向),時間マスキング(横方向).

2,　学習率段階的引き下げ(イテレーション数が、1万回、１０万回に達したタイミングでそれぞれ学習率を0.1倍する。)
